Desarrollo Web

He desarrollado sistemas web con Symfony por años. Symfony es un robusto framework para desarrollo de sistemas web, especial para sistemas comerciales y de alta complejidad. También puedo con otros tipos de frameworks, solo consúltame.

Páginas Web

Drupal es quizás el CMS más versátil que hay, y por esta razón es también el más difícil de aprender. Puedo implementar un sistema con Drupal e incluso diseñar tu página sin plantillas. También puedo con otros CMS.
